Under Review

An IoT-Enabled Lean 4.0 Framework for Learning-Based Process Difficulty Standardization and Difficulty-Adjusted Operator Skill Evaluation in Garment Sewing Lines

Abstract: This study proposes a Lean 4.0 analytical framework that integrates Industrial IoT (IIoT) with advanced data modelling to establish objective, dynamic performance standards. The framework employs hyperbolic learning-curve modelling to capture actual worker progression, refined performance indicators using unsupervised anomaly detection (Isolation Forest, LOF, HDBSCAN), and a hybrid MCDM model integrating CRITIC, Entropy, TOPSIS, and VIKOR to calculate a precise Process Difficulty Index. Results demonstrate superior performance compared to traditional methods, supporting equitable evaluations and "right worker to right workstation" strategy.

Keywords: Lean 4.0 IIoT Machine Learning Anomaly Detection MCDM Learning Curves

Under Review

Optimizing Garment Assembly Line Efficiency Using Ranked Positional Weight and Evolutionary Algorithms: A Comparative Analysis

Abstract: This research develops an integrated method for garment assembly line balancing utilizing Ranked Positional Weight (RPW), evolutionary optimization algorithms, and discrete event simulation. Analysis of a 27-operation jacket sewing line employed Genetic Algorithm (GA) and Differential Evolution (DE) for workload optimization. DE achieved superior results with 16 workstations and 77% line efficiency versus GA's 18 workstations and 71%. Validation through simulation demonstrated practical applicability with full precedence constraint compliance and optimized labor utilization.
